# Poikile

**Four Stoic-inspired color themes for VS Code and Cursor — dark & light, warm & vivid.**

Named after the **Stoa Poikile** ("Painted Porch") in ancient Athens, where Zeno of Citium founded Stoicism around 300 BCE. The palette draws from natural, warm-climate pigments — ochre, terracotta, sage, copper, olive, sand — evoking the painted walls of the ancient colonnade.

---

## Variants

Each variant is named after a Stoic concept that reflects its character.

| Variant | Mode | Style | Meaning |
|---|---|---|---|
| **Poikile - Katalepsis** | Light | Vivid | Clear apprehension and grasped truth; high clarity and precision |
| **Poikile - Euthymia** | Light | Warm | Steady well-being and balanced mood; calm, humane light |
| **Poikile - Ataraxia** | Dark | Warm | Undisturbed calm; quiet, low-noise darkness |
| **Poikile - Apatheia** | Dark | Vivid | Freedom from destructive passions; disciplined contrast in darkness |

## Philosophy

| Stoic Principle | Design Decision |
|---|---|
| **Katalepsis** (clarity) | Every token has exactly one unambiguous visual role |
| **Ataraxia** (tranquility) | Muted, desaturated palette that reduces cognitive noise |
| **Prosoche** (attention) | Functions, types, and errors draw the eye; noise recedes |
| **Sophrosyne** (temperance) | ~14 semantic colors — no gratuitous decoration |
| **Endurance** | Warm neutral tones that won't feel dated in five years |

## Install

### From the Marketplace

1. Open **Extensions** in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **"Poikile"**
3. Click **Install**
4. Open **Command Palette** (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Select **Preferences: Color Theme** and choose one:
   - **Poikile - Ataraxia** (dark warm)
   - **Poikile - Apatheia** (dark vivid)
   - **Poikile - Euthymia** (light warm)
   - **Poikile - Katalepsis** (light vivid)

### From VSIX

```bash
# Build the package
npx @vscode/vsce package

# Install locally
code --install-extension poikile-theme-0.1.0.vsix
```

### Manual

1. Clone this repository into your VS Code extensions directory:

```bash
# macOS / Linux
git clone https://github.com/jduartea/poikile-theme.git \
  ~/.vscode/extensions/poikile-theme

# Windows
git clone https://github.com/jduartea/poikile-theme.git ^
  %USERPROFILE%\.vscode\extensions\poikile-theme
```

2. Restart VS Code and select a theme.

## Recommended Settings

For the best experience, add these to your `settings.json`:

```jsonc
{
  // Enable semantic highlighting for richer token colors
  "editor.semanticHighlighting.enabled": true,

  // Bracket pair colorization
  "editor.bracketPairColorization.enabled": true,
  "editor.guides.bracketPairs": "active",

  // Comfortable editing
  "editor.renderWhitespace": "boundary",
  "editor.cursorBlinking": "smooth",
  "editor.cursorSmoothCaretAnimation": "on",
  "editor.smoothScrolling": true,

  // Recommended fonts (pick one)
  "editor.fontFamily": "'JetBrains Mono', 'Fira Code', 'Cascadia Code', 'Source Code Pro', monospace",
  "editor.fontLigatures": true,
  "editor.fontSize": 14,
  "editor.lineHeight": 1.6
}
```

See [`recommended-settings.json`](./recommended-settings.json) for the full set.
